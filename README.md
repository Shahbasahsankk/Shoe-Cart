
# ShoeCart
  E-Commerce application where user can purchase men and women shoes.


## API Reference

Backend REST API is contributed by https://github.com/aswin-code and code is written in Node.js



## Features

- Payment Integration
- Google authentication
- OTP Verification
- Track orders
- Wishlist
- Cart
